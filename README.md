# 🇵🇹 European Portuguese Reference — PLA A1/A2

A free, fast, mobile-friendly grammar and vocabulary reference for learners of **European Portuguese** at A1/A2 level.

Built as a single HTML file — no app to install, no account required, works on any device.

🔗 **Live site:** [purple-hill-04b8ac910.6.azurestaticapps.net](https://purple-hill-04b8ac910.6.azurestaticapps.net)

---

## What's inside

| Section | Contents |
|---|---|
| **Alphabet** | All 26 letters with names, sounds and key rules (H, R, S, X, LH, NH) |
| **Pronunciation** | Vowel and consonant sound tables, European vs Brazilian differences, listening tips |
| **Pronouns** | Personal pronouns, tu vs você vs o senhor, formal/informal address |
| **Articles** | Definite and indefinite articles, mandatory contractions (no Porto, da Rússia, ao cinema…) |
| **Gender** | Noun gender rules by ending, exceptions |
| **Singular/Plural** | Plural formation rules, adjective agreement in gender and number |
| **Questions** | All 9 interrogative pronouns (quem, onde, quando, porque…) with examples, the é que construction, affirmative/negative/interrogative sentence structure |
| **Ser vs Estar** | Full present and past conjugations, when to use each, tricky cases |
| **Verb Patterns** | -AR / -ER / -IR conjugation families with model verbs (andar, escrever, partir) |
| **Core Verbs** | ter, ir, falar, morar, gostar, querer, poder, chamar-se — with conjugations and examples |
| **Numbers** | 0–1,000,000, ordinals, compound number construction |
| **Days & Months** | Days of the week (numbered system explained), months, time expressions |
| **Identity** | Personal ID vocabulary, marital status, countries with nationalities (m/f) and languages |
| **Family** | Nuclear, extended and reconstituted family vocabulary, all masculine/feminine pairs |
| **House & Home** | Rooms, exterior, floor numbering (Portuguese vs North American) |
| **Description** | Physical appearance (hair, eyes, body, face), personality adjectives, agreement rules |
| **Expressions** | Agreement, opinions, emotions, connectors (mas, porque, então…), café vocabulary |
| **Survival Phrases** | Greetings, classroom phrases, politeness — everything needed from Day 1 |
| **Practice** | Write-in questions to practise self-introduction |

## Features

- 🔍 **Search** — searches across all sections simultaneously with match highlighting
- 📱 **Mobile-friendly** — sidebar navigation on desktop, drawer on mobile
- ⚡ **Instant load** — pure HTML/CSS/JS, no frameworks, no blank screen
- 🔗 **Deep links** — every section has its own URL hash (e.g. `#family`, `#questions`)

## About this reference

This reference was built alongside a **PLA — Português Língua de Acolhimento** (A1/A2) course at [NIALP — Associação Intercultural Lisboa](https://nialp.pt), as part of a Portuguese Golden Visa language requirement. The course runs for 150 hours and concludes with an official A2 certificate.

Content is updated after each session to reflect exactly what was covered in class. All pronunciation is given in phonetic brackets `[like this]` throughout.

**Focus:** European Portuguese specifically — pronunciation, vocabulary and examples reflect Portugal, not Brazil.

---

## Pronunciation note

European Portuguese sounds very different from Brazilian Portuguese. Unstressed vowels are heavily reduced or nearly silent — this is why it can sound like Polish or Russian to new learners. The Pronunciation section explains this in detail with practical tips.
